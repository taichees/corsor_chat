## 案A: WordPress 緊急復旧サービス（ログイン不可 / 白画面 / 500エラー）

- 痛み: 「更新した瞬間ログインできない」「真っ白」「500 Internal Server Error」
  → 公開中サイトで売上/問い合わせが止まるため即決されやすい
- 差別化: 最短60分で初動、FTP/DB直接修正、復旧前にDBバックアップ同梱
- 内容:
  - 1プラグイン/テーマ起因の無効化・ロールバック
  - `wp-config.php` / `.htaccess` の修復
  - PHPバージョン不整合の応急対応
  - 復旧レポート（原因・実施内容・再発防止策）
- 価格案:
  - ライト 5,500円（軽微1件 / 24h以内）
  - スタンダード 12,000円（複数要因 / 当日対応 / バックアップ取得）
  - アップセル: 月次アップデート代行 3,300円/月
- NGライン: サーバ契約者本人の管理画面/FTP情報の提示が必須（事前案内）
- タイトル案: 「WordPressが真っ白/ログイン不可を最短60分で復旧します」

## 案B: お問い合わせフォーム スパム・不達対策パック
- ペルソナ: WordPress / STUDIO / ペライチ等でフォーム設置済みの小規模サイト運営者
- 痛み:
  - 英語スパムが毎日届いて本物の問い合わせが埋もれる
  - 送信完了するのに自分宛にメールが届かない（Gmailに振り分けられる等）
- 差別化: 「スパム対策」と「不達調査」をワンセットにしている出品が薄い
- 内容:
  - reCAPTCHA v3 導入 + honeypot（二重防御）
  - 必須項目/文字種バリデーション調整
  - SMTP送信化（WP Mail SMTP等）で到達性改善
  - SPF/DKIM の簡易チェック + 設定ガイド提出
  - テスト送信 3 パターンで到達確認
- 価格案:
  - シングル 6,500円（スパム対策のみ、1フォーム）
  - セット 9,800円（スパム対策 + SMTP化 + 到達レポート）
- タイトル案: 「問い合わせフォームのスパム激減＆メール不達を1日で解決します」

## 案B 詳細運用（本命）

### パッケージ構成
| プラン | 価格 | 対応範囲 | 納期 |
|---|---|---|---|
| シングル | 6,500円 | スパム対策のみ（1フォーム） | 1営業日 |
| セット（本命） | 9,800円 | スパム対策 + SMTP送信化 + 到達レポート | 1〜2営業日 |
| プレミアム | 14,800円 | セット + SPF/DKIM/DMARC 設定代行 + 月1メンテ | 2〜3営業日 |

### 標準フロー（実働1〜1.5時間）
1. ヒアリング（購入直後テンプレ送付、5分）
   - URL / CMS / フォームプラグイン / 症状 / ドメイン / サーバ会社 / 管理画面ID・PW / DNS管理場所
2. 現状調査（10〜15分）
   - 実送信テスト、スパムログ確認、mxtoolboxでSPF/DKIM/DMARC確認、Gmail/Outlook到達確認
3. スパム対策実装（20〜30分）
   - reCAPTCHA v3 組み込み（CF7 / WPForms / 素のHTML）
   - Honeypot フィールド追加
   - バリデーション強化（文字数・URL個数・キーワード除外）
4. SMTP送信化（15〜20分、セット以上）
   - WP Mail SMTP + SendGrid / Brevo / Gmail / サーバSMTP から選定
5. SPF/DKIM/DMARC設定（20分、プレミアムのみ）
6. テスト送信3パターン（Gmail / Outlook / 本来宛先）と到達確認
7. 納品レポート生成（10〜15分、Cursor丸投げ）

### 使用ツール（全て無料枠内で運用可）
- Google reCAPTCHA v3
- Honeypot for Contact Form 7
- WP Mail SMTP + SendGrid / Brevo 無料枠
- mxtoolbox.com、Google Admin Toolbox MessageHeader

### 想定トラブル
- 管理画面にログインできない → ゲスト管理者発行マニュアルを事前送付
- DNS権限がない → プレミアム作業不可、セットへ降格提案
- Gmailが即迷惑判定 → SMTP化で解決
- reCAPTCHA v3 スコア閾値で誤検知 → 0.5→0.3 or v2フォールバック
- Googleフォーム埋め込み → GASアドオンで対応可能（下記参照）

### Googleフォーム向けアドオン（追加メニュー）
- ライト +2,000円: キーワード/URL/honeypot判定 + 通知メール送信
- スタンダード +3,500円: ライト + スパム退避シート + 自動返信 + Slack通知
- 実装: `onFormSubmit` トリガー + GASスクリプト貼付 + 権限承認手順マニュアル納品
- 実働追加30〜40分、Cursorでコード雛形を生成し質問タイトルだけ差し替え

### 真の対象外（限定）
- Formrun / formzu / Tayori など外部SaaSフォーム（拡張不可）
- Apps Script が組織ポリシーで禁止されている Google Workspace 環境
